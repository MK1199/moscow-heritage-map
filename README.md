# Moscow Heritage Map

Интерактивная карта объектов культурного наследия Центрального административного округа Москвы.

**Данные:** [data.mos.ru](https://data.mos.ru), реестр ОКН, набор №530 (2 440 зданий)  
**Стек:** Python · GeoPandas · Mapbox GL JS

---

## Что внутри

```
moscow heritage/
├── code/
│   ├── 01-data-cleaning.ipynb   # очистка данных, экспорт GeoJSON

```

## Ноутбуки

### 01 — Очистка данных
- Загрузка CSV с двухуровневым заголовком
- Фильтрация по округу (ЦАО) и типу объекта (здания)
- Парсинг полигонов и центроидов из JSON-строк
- Стандартизация категорий: Федеральный / Региональный
- Экспорт `okn-cao-polygons.geojson` и `okn-cao-points.geojson`

### 02 — Анализ и визуализация
- Распределение по категориям охраны и районам ЦАО
- Извлечение дат постройки из названий объектов через regex (точные годы + римские века)
- Маппинг на исторические эпохи (XVII–XX вв.)
- Четыре аналитических графика: donut, stacked bar, timeline, heatmap район × эпоха
- Обновление GeoJSON полями `earliest_year` и `epoch`
